## Features

  * CRUD Category
  * CRUD Product
  * CRUD Bag
  * Search Product name, store, category
  * Filter Name, Category, Product And Group ASC or DESC
  * Validation Input
  * MVC Models


## Quick Start

 Install dependencies

```bash
npm install
```

 Run the app

```bash
npm start
```
  View the website at: http://localhost:3000
